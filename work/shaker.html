Shaker is a cocktail manager: a cookbook-type of app where you can store your favourite cocktail recipes and edit them. You can add a new recipe, see the list of all your cocktails, read their recipes and leave review notes.

Shaker is a Ruby on Rails app deployed with Heroku and using Cloudinary to store images.

What I learned

Shaker was my first full stack project on Ruby on Rails and of the many lessons I learned from it, the most important ones involved my newfound understanding of Rails and the MVC pattern, Railsâ€™ git commands, how to implement many to many relationships (and all the validations and associations that go with it) and using Simple Form with associations.

I also got to:

  • do some API seeding for the cocktail ingredients,
  • use Select2 to dynamically filter the results,
  • discover Cloudinary which I used to allow the user to upload an image for each cocktail recipe.
